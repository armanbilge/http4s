30   /** Transform the service to return a timeout response after the given
31     * duration if the service has not yet responded.  If the timeout
32     * fires, the service's response is canceled.
33     *
34     * @note if the service runs uncancelable effects while responding
35     * (e.g. if service uses [[cats.effect.kernel.MonadCancel#uncancelable MonadCancel#uncancelable]] under the hood) and has exceeded
36     * the timeout, then the expected behavior is:
37     * <ul>
38     *   <li> uncancelable effects will be completed naturally (regardless of how long it takes), </li>
39     *   <li> after that, the timeout response will be returned. </li>
40     * </ul>
41     * To get more insights on effect cancelation, dig into the [[cats.effect.kernel.MonadCancel MonadCancel]] documentation.
42     *
43     * @param timeout Finite duration to wait before returning the provided response
44     */
45   def apply[F[_], G[_], A](timeout: FiniteDuration, timeoutResponse: F[Response[G]])(
46       http: Kleisli[F, A, Response[G]]
47   )(implicit F: Temporal[F]): Kleisli[F, A, Response[G]] =
48     http.mapF(F.timeoutTo(_, timeout, timeoutResponse))
